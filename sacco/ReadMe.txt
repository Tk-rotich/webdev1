################Installation Guide##############################################

1. The database file is contained in the folder.
2. Import the database and setup the sacco folder on your localhost.
3. To start with you must login as admin to be able to manage and add staff members, Companies;
4. To login as admin type "localhost/sacco/admin"  
5. Email=admin@mail.com  and   password=password
